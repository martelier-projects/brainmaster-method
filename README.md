# tiim

## Setup

- Add `.env` file
- Add localhost domain to `.env` (default for Eleventy is `http://localhost:8081/`)

## Mock content

To used the mock content:

- Add `.env` file
- Add `MOCK_CONTENT=true` to `.env` file
- Restart `watch` command
